# EventEase

A seamless event registration and check-in platform designed to simplify attendance, automate workflows, and enhance the entire event experience.

## Features

- Smart QR code-based check-in system
- Automated seat assignment
- Email notifications with QR codes
- Admin dashboard for real-time monitoring
- Manual check-in fallback option
- Excel and CSV export capabilities

## Quickstart

1. Clone the repository
```bash
git clone https://github.com/shravanramakunja/EventEase.git
cd EventEase
```

2. Install dependencies
```bash
npm install
```

3. Configure environment variables
Create a `.env` file with your database and email credentials

4. Start the server
```bash
npm start
```

5. Access the application at `http://localhost:3000`

## Tech Stack

- Node.js
- Express.js
- MongoDB
- EJS templating
- Tailwind CSS
- Nodemailer
